**Arkania – Real Estate Installment Tracking Platform**

## Overview

Arkania Developments is a digital platform designed for unit owners and real estate clients, providing a simple and transparent way to track and manage property installments.  
The system enables users to view payment schedules, monitor due installments, access their unit information, and receive timely reminders through a clean and intuitive experience.

Built using **Node.js** and **TypeScript**, Arkania focuses on financial clarity, reliability, and user convenience within the real estate industry.

## My Contributions

As a backend engineer, I contributed to Arkania’s backend architecture and feature development:

- Built **RESTful APIs** using **Node.js** and **TypeScript** for user authentication, unit details, and installment management.
- Implemented installment schedules, payment history, and real-time status endpoints.
- Designed scalable and structured **MySQL** database schemas.
- Added notification logic for reminding users of upcoming or overdue payments.
- Ensured secure access control for both owners and real estate administrators.
- Optimized data fetching and caching to improve dashboard performance and load times.
- Collaborated with product, mobile, and design teams to ensure smooth integration and alignment with business requirements.

## Technologies Used

- **Backend:** Node.js, TypeScript  
- **Database:** MySQL  
- **ORM:** Prisma
- **APIs:** RESTful  
- **Authentication:** JWT
- **Notifications:** FCM / email notifications  

## Key Features

- Installment tracking with due dates and payment history  
- Real-time unit status and financial overview  
- Secure login and user authorization  
- Automatic reminders for upcoming or overdue payments  
- Dashboard for managing installments and property information  
- Scalable backend ready for future real estate services

## Impact

- Increased transparency for property owners through instant access to financial data  
- Reduced customer support load by centralizing installment and unit information  
- Enhanced user engagement and on-time payments via automated reminders  
- Delivered a modern digital experience for Arkania's real estate clients
